This repository contains scripts to preprocess and reformat (into BIDS) the "The Brain, Body, and Behavior Dataset" (BBBD), 
as well as the scripts to reproduce figures used in the BBBD paper.

Dataset can be downloaded here https://zenodo.org/records/19241964
Website describing the dataset and giving further download options can be found here https://doi.org/10.15387/fcp_indi.retro.bbbd
Paper associated with the dataset can be found here https://www.biorxiv.org/content/10.1101/2025.04.29.651259v1.abstract
